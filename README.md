# Middar Lead Landing

صفحة هبوط مستقلة لجمع الليدز لمنصة Middar.

## الملفات

- `index.html`: محتوى صفحة الهبوط.
- `styles.css`: الثيم والتنسيق المتوافق مع هوية Middar.
- `script.js`: التحقق من النموذج وإرسال البيانات.
- `assets/logo.png`: شعار Middar.

## التشغيل المحلي

افتح `index.html` مباشرة أو شغل خادم ثابت بسيط:

```bash
python -m http.server 4273
```

ثم افتح:

```text
http://127.0.0.1:4273
```
